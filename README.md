# iemeci-web

## テーブルバックアップファイルフォーマット

- 区切り文字: `\t`
- 行区切り文字: `\n`
- ファイル名: `環境名_スキーマ名_テーブル名（_for_用途）-yyyy-mm-dd.tsv`

### 環境名

- aws、herokuの場合のみ付与する
- ローカルの場合のみ省略可能

### スキーマ名

- 必須
- テーブルのスキーマ名とする

### テーブル名

- 必須
- テーブルの名前とする

### for_用途

- 特定の用途がある場合のみ

| 用途 | 用途名 |
| Webステージング用 | _for_stg |

### yyyy-mm-dd

- エクスポート時の日付
